# Changelog

All notable changes to the Scrape project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- AI-powered content analysis with OpenAI
- Export functionality (JSON, CSV, TXT)
- Comprehensive data extraction (images, headings, meta tags, social tags)
- Modern UI with Tailwind CSS
- Download buttons for data export
- AI summary, key points, and category detection

### Changed
- Migrated from plain CSS to Tailwind CSS
- Enhanced scraping to extract more data types
- Improved UI/UX with modern design
- Updated application name to "Scrape"

### Security
- Environment variables for API keys (no hardcoding)
- Enhanced .gitignore to exclude sensitive files

### Added
- Initial release
- Basic web scraping functionality
- Safety features (robots.txt respect, login detection)

- React frontend
- Comprehensive data extraction
- AI-powered content analysis
- Export functionality
- Modern UI with Tailwind CSS
